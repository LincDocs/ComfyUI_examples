# 放大模型示例

以下是如何使用 ESRGAN 等高级模型的示例。将它们放在 models/upscale_models 文件夹中，然后使用 UpscaleModelLoader 节点加载它们，并使用 ImageUpscaleWithModel 节点使用它们。

![Example](esrgan_example.png)
