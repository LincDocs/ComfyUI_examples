# 3D 示例

## Stable Zero123

稳定的 Zero123 是一个扩散模型，给定一个包含物体和简单背景的图像，可以从不同角度生成该物体的图像。

[检查点可以从此处下载，](https://huggingface.co/stabilityai/stable-zero123/blob/main/stable_zero123.ckpt)将其放在 ComfyUI/models/checkpoints 文件夹中。

您可以下载此图像并加载或将其拖到[ComfyUI](https://github.com/comfyanonymous/ComfyUI)上以获取工作流程。

![Example](stable_zero123_example.png)

[输入图像可以在这里](https://comfyanonymous.github.io/ComfyUI_examples/hypernetworks/hypernetwork_example_output.png)找到，它是[超网络](https://comfyanonymous.github.io/ComfyUI_examples/hypernetworks)示例的输出图像。

仰角和方位角以度为单位，控制物体的旋转。
